# 🌾 水稲 葉色診断（荒木農園版）

スマートフォンのカメラを使った、コシヒカリの簡易葉色診断アプリです。
18%反射板で明るさを補正し、葉の写真から**葉色板値・SPAD値・施肥判断の目安**をリアルタイムで表示します。

**開発者**: @boropin

## 📖 使い方

### ステップ 1️⃣ : 生育段階を選択
ホーム画面で、現在の稲の生育段階を選択します。
- **実測のみ** - 数値の記録・比較のみ
- **最大分げつ期** - 6月中旬（参考表示）
- **中干し終了直前** - 6月末～7月初頭（**穂肥判断の主要時期**）
- **出穂期** - 7月中～下旬（追肥判断）
- **収穫直前** - 8月下旬（参考表示）

### ステップ 2️⃣ : 反射板を撮影
1. 屋外の自然光下で、18%反射板をカメラの枠いっぱいに収めます
2. 📷 撮影ボタンをタップ
3. RGB値が自動取得され、明るさの基準が設定されます

**📌 コツ**
- 朝日や西日を避け、太陽が高い時間帯（10時～15時）がベスト
- 反射板を水平に持ち、影がかからないようにしてください
- 撮影後に別の反射板データを選択するまで、この基準が保持されます

### ステップ 3️⃣ : 葉を撮影
1. 測定する葉（上から 2 番目の完全に開いた葉）を撮影
2. **焦点距離**: 葉の中央部が画面を占める程度（10～20cm）
3. 📷 撮影ボタンをタップ

**📌 測定対象の葉**
- **基準**: 上から 2 番目の完全に開いた葉（第 2 展開葉）
- **部位**: 葉の中央部が基準
- **向き**: できるだけ垂直に撮影

### ステップ 4️⃣ : ポイント指定
1. 撮影した葉の写真が表示されます
2. **3 つのポイント** が自動配置されています（赤い円）
3. ポイントをドラッグして、葉の色が濃い部分に合わせます

**📌 ポイント配置のコツ**
- 3 点は葉の異なる部分に配置（色ムラを平均化するため）
- 葉脈や虫食いを避ける
- できるだけ葉の緑色が濃い箇所を選ぶ

### ステップ 5️⃣ : 解析実行
1. 3 つのポイントが配置されたら、「解析開始」をタップ
2. 葉の色を分析し、数秒で結果が表示されます

### ステップ 6️⃣ : 結果確認
以下が表示されます：

| 項目              | 意味                                          |
|-------------------|-----------------------------------------------|
| **葉色板（推定）** | 水稲用葉色板 1～7 の換算値。**4.5 が穂肥の境目** |
| **SPAD値（推定）** | 葉緑素計の相当値。適正は 32～35                 |
| **ExG値**         | グリーン指数。値が高いほど葉が濃い               |
| **推定窒素濃度**   | 参考値（%）                                    |
| **診断・推奨**     | 生育段階に応じた施肥判断の目安                  |
| **位置情報**       | 撮影地点の GPS 座標（許可時のみ）               |

### ステップ 7️⃣ : 複数葉・複数圃場の連続測定
1. 「次の葉を撮影」をタップすると、カメラに戻ります
2. 手順 3～6 を繰り返してください
3. **反射板データは保持** されるので、撮り直す必要はありません

### ステップ 8️⃣ : データ保存
1. 結果画面の「データ保存」をタップ
2. **CSV ファイル** がローカル環境にダウンロードされます。サーバー側には一切残りません。
3. Excel などで開いて、複数回の測定結果を比較できます

## 🔧 機能

### 📷 カメラ機能
- ✅ フロント/バックカメラの切り替え
- ✅ 自動明るさ調整（18%反射板基準）
- ✅ リアルタイムガイド表示（カメラ枠）

### 🎯 画像解析
- ✅ **自動 3 点配置** - 最初から葉の中央 3 箇所に測定点を設置
- ✅ **ズーム機能** - 詳細な位置調整用（拡大率 1～5 倍）
- ✅ **タッチ操作** - ドラッグで自由に移動、ダブルタップで拡大
- ✅ **リアルタイムプレビュー** - 各ポイントの RGB 値を表示

### 📊 診断機能
- ✅ **ExG 値（グリーン指数）** - 葉色の濃さを数値化
- ✅ **SPAD 値推定** - 葉緑素計相当の値
- ✅ **葉色板換算** - 水稲用標準葉色板（1～7）への対応
- ✅ **生育段階別診断** - 最大分げつ期・穂肥期・出穂期に応じた判定

### 🗓️ 生育段階別の評価

| 生育段階           | 時期        | 診断ポイント                                   |
|-------------------|-------------|-----------------------------------------------|
| **実測のみ**       | 通年        | 数値記録・比較                                 |
| **最大分げつ期**   | 6月中旬      | 標準 SPAD31～35 との比較（参考）               |
| **中干し終了直前** | 6月末～7月初  | **穂肥判断の主時期**。葉色板 4.5 が施用の分岐点 |
| **出穂期**        | 7月中～下旬   | SPAD33 未満で追肥検討                         |
| **収穫直前**      | 8月下旬       | 参考値のみ（診断対象外）                       |

### 💾 データ管理
- ✅ **自動保存** - 測定ごとにアプリ内に蓄積
- ✅ **CSV 出力** - Excel で分析可能
- ✅ **複数フィールド対応** - 同一反射板基準で複数圃場を測定
- ✅ **位置情報記録** - GPS 座標と Google Maps リンク

### 📱 対応環境
- ✅ **PWA 対応** - オフラインでも使用可能
- ✅ **モバイル最適化** - スマートフォン専用設計
- ✅ **iOS/Android 対応** - Chrome、Safari で動作
