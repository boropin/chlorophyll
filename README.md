### 本アプリについて
- **参考ツールです**：葉色診断は簡易的な判断支援ツールにすぎません
- **推定値です**：表示される葉色板値・SPAD値・窒素濃度は推定値です
- **カメラ性能に依存**：撮影環境・カメラの性能により精度が変動します

### 施肥判断について
最終的な施肥判断は、以下を総合的に判断して **ご自身の責任で** 決定してください：
- 草丈・茎数
- 天候・気象予報
- 生育時期・圃場の生育状況
- 土壌診断結果
- 過去の栽培経験

## 📚 参考資料・謝辞

本アプリケーション開発にあたり、以下の資料・技術指導を参考にさせていただきました。

### 稲作管理・施肥診断
- **津地域農業改良普及センター** - コシヒカリ穂肥基準（葉色板値4.5の施用判定基準）
- **千葉県農林総合研究センター** - 稲作標準技術体系（SPAD値と葉色板の対応式）
- **農業・食品産業技術総合研究機構（NARO）** - 水稲の窒素診断技術

### 画像解析技術
- **ExG（Excess Green Index）** - 植物の色指数に関する研究
- **NGRDI（Normalized Green Red Difference Index）** - 緑色指数の推定

### 設計・開発
- **Progressive Web App（PWA）技術** - オフライン対応のため
- **Web Camera API** - スマートフォンカメラ連携
