# 参考資料・引用元の詳細

## 1. 稲作管理・施肥診断

### 1.1 穂肥診断基準
**出典**: 津地域農業改良普及センター
- **基準**: コシヒカリ葉色板 4.5 が穂肥施用の分岐点
- **時期**: 中干し終了直前（幼穂形成～減数分裂期）
- **参考時期**: 6月末～7月初頭

### 1.2 SPAD値と葉色板の対応
**出典**: 千葉県「稲作標準技術体系」
- **式**: SPAD = カラースケール × 7.8 + 1.6
- **逆算**: カラースケール = (SPAD − 1.6) / 7.8

### 1.3 出穂期の窒素診断
**出典**: 農業・食品産業技術総合研究機構（NARO）
- **基準**: SPAD 33 以上が登熟期に向けた適正値
- **高温対策**: 登熟期高温予想時は SPAD33 未満で追肥を検討

## 2. 画像解析手法

### 2.1 ExG（Excess Green Index）
植物の緑色を抽出する指数
- **式**: ExG = 2*G - R - B
- **応用**: 葉の色濃度を数値化

### 2.2 NGRDI（Normalized Green Red Difference Index）
正規化された緑・赤差指数
- **式**: NGRDI = (G - R) / (G + R)
- **応用**: 窒素濃度の推定に使用
